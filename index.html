Title:  You Got OptionsTM

Objective:  To demonstrate the coding skills I am learning in LC101 using HTML, CSS, Python, and a database, all wrapped up in a framework (e.g., GAE, django). Click here for more. Equally important, this web app will allow the visitor to chart the historical performance of select options strategies.

Scope:  This web app will show time-series data visualization (aka charts) of certain options, using end-of-day data downloaded from the internet. This data and these charts are basically meaningless to someone who does not understand options, and I will make no attempt to educate the viewer about options; there are many, many resources on the internet available for that.

Features:  Dynamic charts showing the historical performance of select options trading strategies.

  • Charts  These charts were produced in October 2016 from options data I had loaded into a MySQL database. Unfortunately, that database was on a Windows computer which has since stopped working. I then installed Linux on that same computer, and expect better results in the future.

Benefits:  Allows options trader to see and analyze the historical performance of certain options strategies.

Methods:  I will need to build a database like MySQL, PostgreSQL or SQLite, to feed the data into a charting package like gnuplot or matplotlib.

Summary:  This is a work in progress. Future iterations will include features I learn in LC101. Also, the research abstract outline will disappear (css display:none) as the web page matures.

Disclaimer:   The intended audience for this web app is myself, my LC101 instructors, and potential employers, and no one else. The SEC frowns upon unlicensed and uncertified options traders giving financial advice, therefore, no part of this website is intended to be or may be construed to be options trading recommendations.
