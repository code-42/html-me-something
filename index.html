Title: You Got OptionsTM

Objective:  To demonstrate the coding skills I am learning in LC101 using HTML, CSS, Python, and a database, all wrapped up in a framework (e.g., GAE, django). Click here for more. Equally important, this web app will allow the visitor to chart options strategies.

Scope:  This web app will show time-series data visualization (aka charts) of certain options, using end-of-day data downloaded from the internet.

Features:  Dynamic charts showing the historical prices of options trading strategies

Charts

Benefits:  Allows options trader to see the performance history of certain options

Methods:  I will need to build a database like MySQL, PostgreSQL or SQLite.

Summary:  This is a work in progress. Future iterations will include features I learn in LC101. Also, the research abstract outline will disappear as the web page matures.

Disclaimer:   The intended audience for this web app is myself, my LC101 instructors, and potential employers, and no one else. The SEC frowns upon unlicensed and uncertified options traders giving financial advice, so no part of this website is intended or may be construed to be options trading recommendations.
